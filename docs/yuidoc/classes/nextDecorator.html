nextDecorator Class

Uses
  • Button
Defined in: scripts/components/Button/decorators/next.js:3
Module: Lu

Decorates the Button to listen for the 'selected' event and to intelligently disable or enable itself based upon the control having a next item. This is used in conjunction with the default decorator.
